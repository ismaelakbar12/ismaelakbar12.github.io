Web Scrapping dengan rvest pada situs Jaringan Keselamatan Penerbangan

Penulis : Ismail Akbar | Prof. Dr. Suhartono M.Kom | Magister Informatika | UIN Maulana Malik Ibrahim Malang | https://ismaelakbar.site/

06 Juni 2021

A. Pengantar

Data dan informasi di web saat ini sangat berkembang pesat. Kita semua saat ini mengakses website sebagai sumber pengetahuan pertama kita, baik itu tentang menemukan ulasan tentang suatu tempat hingga memahami istilah baru. Semua informasi ini sudah tersedia di web.

Web scraping adalah proses mengekstrak data dari sebuah situs web. Salah satu contoh web scraping adalah menyalin daftar kontak dari direktori web.

Web scraping akan dilakukan di situs web aviation-safety.net. Data Jaringan Keselamatan Penerbangan akan diambil untuk mengetahui data kejadian kecelakaan penerbangan yang ada di dunia dengan memanfaatkan library rvest.

Package rvest ditulis oleh Hadley Wickham dari RStudio. Package ini mempunyai fungsi yang serupa dengan library beautiful soup pada Python, yaitu untuk web scraping. Langkah pertama, tentu saja, install dan load package rvest.

C. Membaca Data Bagian 1

Pada kasus ini kita ingin membaca semua insiden yang terjadi pada tahun 2020 dengan mengakses alamat web aviation-safety.net/database/dblist.php?Year=2020. Akan tetapi pada halaman website tersebut menampilkan data frame lebih dari satu pada setiap tahunnya. Untungnya kita bisa mendapatkan jumlah halaman dengan menggunakan rvest juga.

url <- "http://aviation-safety.net/database/dblist.php?Year=2020"
pages <- url %>% html() %>%
  html_nodes(xpath = '//*[@id="contentcolumnfull"]/div/div[2]') %>%
  html_text() %>% strsplit(" ") %>% unlist() %>%
  as.numeric() %>% max()

Sekarang kita dapat menulis perulangan kecil untuk mendapatkan semua insiden pada tahun 2020, karena tautan web tersebut akan berubah dengan nomor halaman.

# memulai data frame kosong, di mana kami akan menyimpan data
data <- data.frame(date = numeric(0), type = numeric(0), registration = numeric(0),
                   operator = numeric(0), fatalities = numeric(0), location = numeric(0), 
                   category = numeric(0))
# loop melalui semua nomor halaman
for (page in 1:pages){
  # buat URL baru untuk halaman saat ini
  url <- paste0("http://aviation-safety.net/database/dblist.php?Year=2020&lang=&page=", page)
  # mendapatkan data html dan ubah menjadi data.frame
  incidents <- url %>% html() %>%
    html_nodes(xpath = '//*[@id="contentcolumnfull"]/div/table') %>%
    html_table() %>% data.frame()
  # menggabungkan data
  data <- rbind(data, incidents)
}

Bagaimana caranya kita mendapatkan data selama beberapa tahun? Maka kita perlu mengatur nilai tahun awal dan tahun akhir.

# pengaturan nilai awal
startyear <- 2018
endyear <- 2020
url_init <- "http://aviation-safety.net/database/dblist.php?Year="

dat <- data.frame(date = numeric(0), type = numeric(0), registration = numeric(0),
  operator = numeric(0), fatalities = numeric(0),
  location = numeric(0), category = numeric(0))
for (year in startyear:endyear){
  # dapatkan url untuk tahun ini
  url_year <- paste0(url_init, year)
  # dapatkan halaman
  pages <- url_year %>% html() %>%
    html_nodes(xpath = '//*[@id="contentcolumnfull"]/div/div[2]') %>%
    html_text() %>% strsplit(" ") %>% unlist() %>%
    as.numeric() %>% max()
  # ulangi melalui halaman
  for (page in 1:pages){
    url <- paste0(url_year,"&lang=&page=", page)
    # mendapatkan data html dan ubah menjadi data.frame
    incidents <- url %>% html() %>%
      html_nodes(xpath = '//*[@id="contentcolumnfull"]/div/table') %>%
      html_table() %>% data.frame()
    # menggabungkan data
    dat <- rbind(dat, incidents)
  }
}
